## 二、启动中间件（Docker）

项目依赖三个服务，用 Docker Compose 一键启动。

### 首次启动

```bash
cd D:/DJ/agent学习/个人项目/multi-tenant-agent
docker compose up -d
```

### 确认状态

```bash
docker compose ps
```

正常输出：

| 服务           | 镜像                   | 端口        | 状态      |
| ------------ | -------------------- | --------- | ------- |
| mta-postgres | postgres:16          | 5432      | healthy |
| mta-qdrant   | qdrant/qdrant:latest | 6333/6334 | running |
| mta-redis    | redis:7              | 6379      | healthy |

### 停止中间件

```bash
docker compose down
```

### 清除数据重新开始

```bash
docker compose down -v      # 删除数据卷（数据会丢）
docker compose up -d         # 重建容器
cd backend
alembic upgrade head         # 重建数据库表
```

> `-v` 会删除数据卷，数据库数据会丢失，必须重新执行 Alembic 迁移。

## 三、启动后端（FastAPI）

### 激活虚拟环境

```bash
cd D:/DJ/agent学习/个人项目/multi-tenant-agent/
.\venv\Scripts\Activate.ps1
```

看到提示符前面出现 `(venv)` 即成功。

### 启动服务

```bash
cd backend
uvicorn app.main:app --reload
```

- `--reload`：代码修改后自动重启，开发用
- 默认监听 `http://127.0.0.1:8000`

### 访问地址

| 地址                             | 用途                 |
| ------------------------------ | ------------------ |
| <http://127.0.0.1:8000/health> | 健康检查               |
| <http://127.0.0.1:8000/docs>   | API 文档（Swagger UI） |
| <http://127.0.0.1:8000/redoc>  | API 文档（ReDoc）      |

## 三-2、中间件可视化

### PostgreSQL

通过 VS Code 插件 **Database Client** 连接，连接信息：

| 字段       | 值              |
| -------- | -------------- |
| Host     | `localhost`    |
| Port     | `5432`         |
| Database | `mta_db`       |
| User     | `mta_user`     |
| Password | `mta_password` |

### Qdrant（向量数据库）

Qdrant 自带 Web 控制台，浏览器直接访问：

```
http://localhost:6333/dashboard
```

功能：

- 查看 Collections（向量集合）
- 浏览已存储的向量数据
- 执行搜索测试
- 查看 Collection 的统计信息

### Redis

可通过 VS Code 插件或命令行查看：

```bash
docker exec -it mta-redis redis-cli
> keys *
> get <key>
> exit
```

## 四、数据库管理

### 执行迁移（表结构变更后）

```bash
cd D:/DJ/agent学习/个人项目/multi-tenant-agent/backend
alembic upgrade head
```

### 创建新迁移（修改模型后）

```bash
alembic revision --autogenerate -m "描述信息"
```

### 回滚迁移

```bash
alembic downgrade -1    # 回滚一步
alembic downgrade base  # 回滚全部
```

### 查看当前版本

```bash
alembic current
```

## 七、常见问题

### Q: Docker 拉取镜像超时

配置镜像加速器：Docker Desktop → Settings → Docker Engine → `registry-mirrors`：

```json
"registry-mirrors": [
  "https://docker.1ms.run",
  "https://docker.xuanyuan.me"
]
```

### Q: bcrypt 报错 `module 'bcrypt' has no attribute '__about__'`

版本不兼容，执行：

```bash
pip install bcrypt==4.0.1
```

### Q: Pylance 报 `无法解析导入`

VS Code 左下角点击 Python 版本 → 选择 `venv` 虚拟环境。

### Q: 新终端里 python/pip 命令不对

需要先激活虚拟环境：

```bash
.\venv\Scripts\Activate.ps1
```

### Q: 终端卡住无法输入（uvicorn 未正常退出）

强制杀掉所有 Python 进程：

```bash
taskkill /F /IM python.exe
```

然后重新打开终端启动服务。

### Q: uvicorn Ctrl+C 时报 CancelledError
